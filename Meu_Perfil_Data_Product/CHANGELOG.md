# Changelog

Todas as alterações notáveis neste projeto de CV serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.1.0] - 2025-10-09

### Alterado
- Atualizado resumo profissional para primeira pessoa em todos os arquivos principais:
  - CV_compilado.md (português)
  - CV_compilado_en.md (inglês)
- Refinado tom geral do sumário para melhor refletir personalidade profissional

## [1.0.0] - 2024-05-14

### Adicionado
- Versão inicial completa do CV com dados reais
- Estrutura de pasta organizada em seções (Resumo, Experiência, Formação, Publicações)
- Versões consolidadas em português e inglês do CV
- GitHub Action para verificação automática de TODOs

### Corrigido
- Resolvidos todos os problemas de encoding e caracteres especiais em português
- Substituídos todos os placeholders por conteúdo real
- Corrigidos links e referências cruzadas entre arquivos

### Alterado
- Organização de experiência profissional em ordem cronológica decrescente
- Padronização de formatos de datas e períodos em todos os arquivos
- Normalização de nomes de empresas e títulos de cargos
