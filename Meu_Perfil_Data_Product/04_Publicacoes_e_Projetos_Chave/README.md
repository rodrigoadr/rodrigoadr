# Publicações e Projetos Chave

Esta seção contém projetos públicos, publicações técnicas, contribuições open-source e repositórios relevantes.

## Estrutura sugerida

Crie arquivos separados para cada tipo de conteúdo:
- Projetos pessoais: `YYYY_Nome_Projeto.md`
- Artigos e publicações: `Artigos.md` (lista consolidada com links)
- Contribuições open-source: `Open_Source_Contributions.md`
- Palestras e apresentações: `Palestras.md`
- Portfólio técnico: `Portfolio.md` (projetos demonstráveis)
