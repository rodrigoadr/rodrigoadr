# Data Product Specialist — Grupo Boticário

**Empresa:** Grupo Boticário  
**Cargo:** Especialista em Produto de Dados (Data Product Specialist)  
**Período:** Março de 2023 – Presente  
**Local:** São José dos Pinhais, Paraná, Brasil (Híbrido)  
**Acesso ao cargo via:** Promoção interna (Especialista de Domínio → Data Product Specialist)

---

## Contexto

Cargo de maior senioridade na jornada de dados do Grupo Boticário, liderando a estratégia e execução de produtos de dados para o ecossistema de franquias, operations e marketing da companhia.

## Responsabilidades Principais

- Liderança do roadmap de produtos de dados estratégicos para o ecossistema de mais de 500 franquias.
- Design de arquitetura de Data Mesh por domínios com contratos de dados versionados.
- Aplicação de ML e GenAI em produção: pioneiro na criação do primeiro LLM da companhia para classificação automática de incidentes via Vertex AI.
- Democratização de acesso a dados para franqueados via plataformas self-service em GCP (BigQuery + Looker).
- Gestão de governança e qualidade de dados em ambiente multi-domínio com pipelines DataOps.
- Colaboração com lideranças de negócio (C-Level e Diretores) para definição de OKRs orientados a dados.

## Principais Conquistas

- Migração de ~56 TB de dados analíticos de QlikView/Qlik Sense para GCP utilizando arquitetura medallion, sem perda de governança ou continuidade de negócio.
- Deploy de modelos de ML e LLM em produção via Vertex AI e BigQuery ML.
- Plataforma de dados acessada por 500+ franqueados com disponibilidade e performance SLA definidos.
- Redução de ~40% no tempo de resposta a incidentes de dados com classificação automatizada.

## Stack Tecnológica

- **Cloud:** GCP — BigQuery, Vertex AI, Dataflow, Cloud Composer, Looker
- **Machine Learning:** BigQuery ML, Vertex AI, LLM (classificação de incidentes)
- **Orquestração:** Apache Airflow (Cloud Composer)
- **Linguagens:** Python, SQL
- **Governança:** Dataplex, Data Catalog, contratos de dados
- **Observabilidade:** DataOps com monitoramento de qualidade de dados
