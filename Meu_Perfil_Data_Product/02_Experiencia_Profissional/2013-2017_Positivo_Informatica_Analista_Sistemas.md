# Positivo Informática — Analista de Sistemas (2013–2017)

**Empresa:** Positivo Informática S.A.  
**Cargo:** Analista de Sistemas  
**Período:** Agosto de 2013 – Fevereiro de 2017  
**Local:** Curitiba, Paraná, Brasil

---

## Resumo

Atuação no P&D de soluções backend e banco de dados para plataformas educacionais digitais da Positivo Informática, uma das maiores empresas de tecnologia educacional do Brasil. Desenvolvimento do backend de educação adaptativa e integração com padrão xAPI/TinCan para rastreamento de aprendizagem.

## Responsabilidades Principais

- Desenvolvimento de soluções backend (Java/.NET) para plataformas de ensino a distância (EAD)
- Construção e manutenção de LRS (Learning Record Store) utilizando protocolo xAPI (TinCan)
- Desenvolvimento do backend do sistema de educação adaptativa
- Criação de livros digitais interativos para uma das maiores redes de ensino do Brasil
- Modelagem de banco de dados relacional e NoSQL para aplicações educacionais
- Integração com plataformas de conteúdo e sistemas de gestão de aprendizagem (LMS)
- Participação em revisões de arquitetura e definição de padrões técnicos

## Principais Conquistas

- LRS baseado em xAPI implementado e em uso por centenas de escolas parceiras
- Backend de educação adaptativa suportando personalização de conteúdo por perfil de aluno
- Participação em produto que impactou dezenas de milhares de alunos nas redes escolares parceiras
- Evolução da arquitetura de EAD para suportar escala nacional

## Stack Tecnológica

- **Backend:** Java, .NET (C#), RESTful APIs
- **Bancos de Dados:** PostgreSQL, MySQL, MongoDB
- **Protocolos:** xAPI (TinCan), SCORM, IMS
- **Plataformas:** LMS (sistemas de gestão de aprendizagem)
- **Ferramentas:** Git, Maven, Jenkins
