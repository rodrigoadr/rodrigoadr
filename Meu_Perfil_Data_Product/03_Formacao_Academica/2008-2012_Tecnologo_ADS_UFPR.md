# 2008–2012 Tecnologia em Análise e Desenvolvimento de Sistemas — UFPR

## Período
Mar/2008 — Dez/2012

## Instituição
Universidade Federal do Paraná (UFPR)

## Tipo
Graduação — Tecnólogo

## Descrição
Formação técnica em desenvolvimento de software, algoritmos, estruturas de dados e engenharia de software. Curso focado na formação de profissionais para o mercado de desenvolvimento de sistemas.

## Principais disciplinas
- Programação orientada a objetos (Java, C++)
- Estruturas de dados e algoritmos
- Banco de dados relacionais e SQL
- Engenharia de software e análise de requisitos
- Desenvolvimento web (HTML, CSS, JavaScript, PHP)
- Redes de computadores e sistemas distribuídos
- Arquitetura de software e padrões de design
- Gestão de projetos de software

## Competências desenvolvidas
- Desenvolvimento de aplicações desktop e web
- Modelagem e implementação de bancos de dados
- Análise de requisitos e design de sistemas
- Programação em múltiplas linguagens
- Fundamentos de redes e infraestrutura
- Metodologias de desenvolvimento de software

## Projetos acadêmicos
- Sistema de gestão acadêmica (projeto integrador)
- Aplicação web para e-commerce (trabalho de conclusão)
- Sistema de controle de estoque com Java Swing
- Projeto de banco de dados para biblioteca universitária

## Tecnologias utilizadas
- Java, C++, PHP, JavaScript
- MySQL, PostgreSQL
- Eclipse, NetBeans
- Apache, Tomcat
- HTML, CSS, XML
